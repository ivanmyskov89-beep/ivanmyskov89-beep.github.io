📍 IP Detector ⭐ Ключевой проект

Профессиональный инструмент для определения геолокации по IP-адресу с автоматической загрузкой результатов в облачное хранилище.

  • 🔍 Определение публичного IP через ipify.org
  • 🌍 Получение геоданных (город, страна, координаты) через ipinfo.io
  • ☁️ Автоматическая загрузка JSON-результатов на Яндекс.Диск
  • 💻 Три интерфейса: CLI, Web, REST API
  • 🐳 Docker контейнеризация
  • 📊 Подробное логирование с Loguru
  • 🧪 Юнит-тесты с покрытием 85%

📱 Мои Telegram-боты

🎨 AI Image Bot

Генерирует уникальные изображения по текстовому описанию с помощью нейросети

Как работает:

  • ✍️ Отправляете текстовое описание боту
  • 🎨 Нейросеть Pollinations.ai генерирует изображение
  • ⭐ Есть бесплатные попытки (5 в день)
  • 💎 Платные подписки через Telegram Stars (больше генераций, безлимит)
  • 📤 Бот отправляет готовую картинку обратно

Ключевая особенность: Полностью автоматическая монетизация — пользователи могут покупать генерации прямо в Telegram.
